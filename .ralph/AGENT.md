# Ralph Agent Configuration

## Build Instructions

```bash
# Install dependencies
npm install

# Build the project
npm run build
```

## Test Instructions

```bash
# Run tests
npm test
```

## Run Instructions

```bash
# Install dependencies (if not already installed)
npm install

# Start development server
npm run dev

# Preview production build
npm run preview
```

## Notes
- This is a React + Vite project with Three.js for 3D rendering
- Development server runs on http://localhost:3000
- Environment variables should be placed in .env file
- Update this file when build process changes
