# 刘语悦周岁宴 - 3D星云式请柬网页设计文档

## 一、项目概述

### 1.1 项目定位
为刘语悦宝宝的周岁宴打造一个极具视觉冲击力的3D互动请柬网页，通过星云式的照片浏览体验，让来宾在探索中感受宝宝成长的点滴。

### 1.2 核心体验
- **太空探索感**：照片如星辰般漂浮在3D空间中
- **丝滑交互**：手指滑动带有惯性和阻尼效果
- **沉浸式预览**：点击放大查看照片细节
- **便捷回复**：一键填写飞书文档确认出席

## 二、视觉设计方案

### 2.1 整体风格定位
- **主题**：梦幻宇宙 × 温馨童趣
- **色调**：深邃星空蓝（#0a1628）+ 温暖粉橙渐变（#ff6b9d → #ffa06b）
- **氛围**：科技感 + 柔和童真

### 2.2 核心视觉元素

#### 2.2.1 背景层
```
- 深色星空背景（粒子系统模拟星星闪烁）
- 渐变光晕（径向渐变，从中心向四周扩散）
- 流动星云（使用噪声算法生成动态云雾效果）
```

#### 2.2.2 照片星云层
```
布局方式：球状分布（Spherical Layout）
- 中心区域：3-5张核心照片（宝宝标志性照片）
- 中间层：10-15张成长照片
- 外围层：5-10张生活花絮

尺寸策略：
- 近处照片：180-240px（手机端）
- 中间照片：120-160px
- 远处照片：80-100px
```

#### 2.2.3 UI层（2D覆盖层）
```
顶部导航区：
- Logo/标题：「语悦一周岁」
- 副标题：「探索我的成长星云」

底部信息卡：
- 半透明毛玻璃材质（backdrop-filter: blur(10px)）
- 渐变边框（粉橙色）
- 内容：时间、地点、飞书回执入口

浮动操作按钮：
- 「确认出席」按钮（右下角悬浮）
```

## 三、功能模块设计

### 3.1 核心功能模块

#### 模块1：3D星云照片浏览器
**功能描述**：
- 几十张照片在3D空间中随机分布
- 支持手指滑动旋转视角
- 惯性滑动效果（阻尼系数可调）
- 自动聚焦最近的照片

**技术实现**：
```javascript
// 核心库
- Three.js (3D渲染引擎)
- React Three Fiber (React集成)
- @react-three/drei (辅助工具库)

// 关键组件
PhotoSphere.jsx    // 照片球体容器
PhotoCard.jsx      // 单张照片3D平面
OrbitControls.jsx  // 自定义滑动控制器
```

**性能优化**：
- 双分辨率策略（缩略图256x256 + 高清原图）
- 懒加载：只加载视野内的照片
- 纹理复用：共享材质和几何体
- 帧率限制：移动端30-60FPS自适应

**动画效果**：

```javascript
  maxDistance: 20,           // 最大缩放距离
```

## 七、性能优化策略

### 7.1 加载优化
1. **渐进式加载**：
   - 首屏只加载3-5张核心照片缩略图
   - 用户滑动时，按需加载视野内的照片
   - 高清原图只在点击放大时加载

2. **图片压缩**：
   - 缩略图：WebP格式，256x256，质量80%
   - 原图：WebP格式，1920x1080，质量85%

3. **代码分割**：
   - 3D场景与UI层分离
   - 按路由懒加载

### 7.2 渲染优化
1. **帧率控制**：
   - 移动端自适应30-60FPS
   - 使用`useFrame`控制渲染频率

2. **几何体复用**：
   - 所有照片共享同一个PlaneGeometry
   - 只有贴图不同

3. **视锥体剔除**：
   - 只渲染视野内的照片
   - 距离超过阈值的照片自动隐藏
