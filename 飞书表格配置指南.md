## 💻 在网页中使用

### 方法1: API调用（推荐）
网页前端通过后端代理调用飞书API,自动提交回执数据。

**流程**：
```
用户填写表单 → 前端发送到后端 → 后端调用飞书API → 数据写入表格
```

**优点**：
- 样式完全自定义
- 更好的移动端体验
- 可以做数据验证

### 方法2: 飞书表单嵌入
如果你觉得API太复杂,也可以直接使用飞书自带的表单功能：

1. 在飞书多维表格中,点击「创建表单」
2. 配置表单字段（称呼、人数等）
3. 获取表单链接
4. 在网页中嵌入 iframe

**示例代码**：
```html
<iframe
  src="你的飞书表单链接"
  width="100%"
  height="600px"
  frameborder="0"
></iframe>
```

**优点**：
- 零代码,开箱即用
- 飞书自动收集数据

**缺点**：
- 样式不可定制
- 移动端体验一般
